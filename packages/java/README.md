# Java Shared Packages

Minimal shared Java package lane with Gradle and JUnit.

- `com.elloloop.scaffold.shared.platform`: cross-service primitives and platform seams.
- `com.elloloop.scaffold.shared.serverkit`: service bootstrap helpers and health contracts.
- `com.elloloop.scaffold.shared.testkit`: deterministic test helpers.

```sh
./gradlew test
./gradlew run
```
